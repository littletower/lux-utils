    • js-library-boilerplate

    js-library-boilerplate is a boilerplate to create a simple Javascript library.

    Features

    • Build : create production-ready distribution file
    • Documentation : create different documentation for each version
    • Lint : check the quality of your code
    • Test : test your code and check the code coverage
    • Example : try your library with examples
    • Continuous Integration : set your continuous integration environment with Travis CI

    Usage

    1. Clone this repository
    git clone https://github.com/docomodigital/js-library-boilerplate.git
    1. Create your library folder, if it does not already exist
    mkdir yourMagicLibrary
    1. Create the library, inserting the path of your library
    cd js-library-boilerplate
node install.js ../yourMagicLibrary
    1. Go to new magic library, create a void README and install dependencies
    cd ../yourMagicLibrary
touch README.md
npm install

    1. Remember to check and replace necessary fields in your package.json

    2. Now you can:

    3. Edit source code and test of your library, contained in src/

    4. Check the quality of your code with npm run lint
    5. Run tests with npm run test
    6. Check the coverage of your tests with npm run test:open
    7. Try your library with the examples contained in examples/, with npm start
    8. Create the documentation and read it with npm run doc:open
    9. Integrate your library with Travis CI, using .travis.yml
    10. Publish your library on NPM

    You can find all commands for develop, maintain and publish your library on CONTRIBUTING.md.
